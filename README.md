# NIFTY 50 Risk & Return Analytics Dashboard

## Project Overview

This project analyzes selected NIFTY 50 stocks using Python and Power BI to evaluate stock performance through risk-adjusted metrics.

The dashboard focuses on identifying top-performing stocks, comparing volatility vs return, and ranking equities using Sharpe Ratio.

## Objectives

* Analyze stock returns using real market data
* Compare risk vs reward across stocks
* Identify top performers
* Build an interactive executive dashboard

## Tools Used

* Python
* Pandas
* NumPy
* yFinance
* Power BI
* Jupyter Notebook

## Key Metrics Used

* Annual Return %
* Volatility %
* Sharpe Ratio
* Total Return %

## Dashboard Features

* KPI cards for quick insights
* Sharpe Ratio ranking chart
* Risk vs Return scatter chart
* Top performing stocks chart
* Metrics overview table

## Files Included

* `NIFTY50_ANALYTICS.pbix`
* `main_project.ipynb`
* `metrics.csv`
* `dashboard.png`

## Key Insight

The analysis showed that some stocks delivered stronger risk-adjusted returns than others, highlighting opportunities where return was attractive relative to volatility.
